## The Assessments

### Stroop Task
The Stroop Task is a neuropsychological test used to examine attention for both clinical and experimental purposes in the field of cognitive neuroscience (MacLeod, 1992). Participants are first presented with a colour name with a coloured font. In some trials, the colour names are congruent as the the coloured fonts they have (eg. blue having a blue font). However, sometimes the colour names are incompatible with the coloured font (eg. blue having a red font). The goal of the task is to have the participant respond with the colour of the font when shown any colour name. For example, in a trial where blue has a red font, participants would have to answer “red”. The Stroop task is a prominent task to measure attention as it places an automatic process (word reading) against a controlled process (colour naming). Performance can be operationalized by the rate and accuracy of responses.<br><br>

### Flanker Task
The Eriksen Flanker task is a set of response inhibition tests used to test the ability to suppress responses that are inappropriate when shown a specific image. In cognitive psychology, the Flanker task is used to measure the spatial/selective attention and executive control of a participant. The Flanker task is useful to help articulate the conceptual and neural models of action control which helps illustrate influential roles of developmental trends in action control, fundamental deficits in action control due to neuropsychiatric disorders, and enhanced action control among elite athletes (Ridderinkhof et al., 2020). This task requires selective attention and inhibition control.<br><br>

### Approximate Number Estimation Task
This task measures the approximate number system (ANS), which is an element of cognition dedicated to number sense and relation (Halberda & Odic, 2015). This task has been used to predict a person’s mathematic capabilities and performance, in correlation with the acuity of their ANS (Cai et al., 2018).<br><br>

### Reading the Mind in the Eyes Test
The Reading the Mind in the Eyes is a popular test used to measure an individual’s emotional intelligence and their ability to recognize emotions in others. This test originated from a study in 1997 intended to help measure autistic adults’ emotional intelligence, particularly when it came to social interactions and eye reading (Baron-Cohen et al 1997). The test itself requires participants to click through a selection of facial images, all of which consist of a person’s eyes. From these images, participants are provided with four choices to correctly describe the emotion they believe to be shown in the eyes of the images. Out of the four options, only one is correct, and the goal is to find the correct adjective for the image. The purpose of these images and options is to test an individual’s ability to read emotions through the eyes, ultimately hoping for high correct scores to equate to high emotional intelligence.<br><br>

### Speech-in-Noise Task
The speech-in-noise task measures someone's ability to identify spoken words when there is background noise and also measures working memory after completing a task. We split up the words that we chose by groups of syllables (we had one, two and three syllable words) and we wanted to know and measure if there was any relationship between the number of syllables and the length of time it took someone to guess the word. Or, if they found it too difficult, we could see if the length of the word relates to how many words they left blank. This task has a wide array of uses ranging from musical training on performance to the development of perception of speech-in-noise. This would be especially helpful when trying to evaluate (to a certain degree) the acquisition of a second language. When an individual is able to identify the word with background noise, it helps demonstrate the degree of their language development and how well they can decipher words in background noise.<br><br>

### Matrix Reasoning Task
The matrix reasoning task is designed as a measure of fluid intelligence. In this task, you will see a matrix of visual patterns. The lower-right part of the pattern will be missing. Your job is to select the best option that completes the pattern. There are 42 total patterns, and you will have 60 minutes to complete it.<br><br>
[[READ THE USER MANUAL]](https://run.pavlovia.org/2750e/matrix-reasoning/Dossier.pdf)<br>

### Go/No-Go Task
This is a classic paradigm for measuring cognitive control. On each trial, you will see one of two letters. One letter signals that you must respond as quickly as possible. The other letter signals that you must refrain from responding. 100 total trials. Expected duration: 6-10 minutes.<br><br>
[[READ THE USER MANUAL]](https://run.pavlovia.org/2750e/go-nogo/Dossier.pdf)<br>

### Attention Network Test
The Attention Network Test (ANT) is a popular assessment of three subtypes of attention: alerting, orienting, and executive. It essentially combines a Flanker paradigm with a Posner cueing task. This specific script has been adapted from the Experiment Factory (expfactory.github.io). Expected duration: 15 minutes.<br><br>
[[READ THE USER MANUAL]](https://run.pavlovia.org/2750e/ANT/Dossier.pdf)<br>

### Simon Memory Game
The Simon Task is based on the popular memory game in which individuals must remember a sequence of tones/colored buttons, which light up in a specific order. It is meant as a measure of short-term memory and attention. You will play 20 "games" of Simon in this script. Expected duration: 60 minutes.<br><br>
